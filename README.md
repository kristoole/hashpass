# Hashpass Hashcracking Web Application 

## Build Setup
```
bundle install
yarn
```

## run development
```
foreman start
# for back-end
# open http://localhost:9292
# for front-end
# open http://localhost:8080
```

## run production
```
yarn run build
APP_ENV=production bundle exec rackup
# open http://localhost:9292
```
